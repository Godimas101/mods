# 🤝 Contributing to SE Mods

Thanks for helping improve `se-mods`.

---

## 🐞 Bugs and compatibility issues

Please open a GitHub issue for:

- broken or outdated mod behavior
- compatibility problems after game patches
- missing documentation
- feature requests for new mod additions

Include the game version, reproduction steps, and mod name when relevant.

---

## 🔀 Pull requests

PRs are welcome.

Please:

1. keep changes focused and easy to review
2. describe exactly which mod or folder is affected
3. update docs when setup or compatibility changes
4. avoid committing local-only notes or scratch files

---

## 👀 Review flow

This repo uses code ownership and auto-assignment so **`@Godimas101`** is requested on pull requests and tagged as the default maintainer contact.
